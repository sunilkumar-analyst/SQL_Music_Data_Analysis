# SQL_Project_Music_Store_Analysis
SQL project to analyze online music store data

📌 **Project Description**
This project involves analyzing a relational music store database using SQL by solving a structured set of business questions ranging from basic to advanced difficulty. The analysis is performed across multiple interconnected tables such as customers, invoices, tracks, artists, and genres to extract meaningful insights.
The project is divided into three levels: Easy, Moderate, and Advanced, demonstrating progressive SQL skills.
🎯 **Key Skills Demonstrated**
•	Multi-table JOIN operations
•	Aggregation functions (SUM, COUNT, AVG)
•	GROUP BY & ORDER BY
•	Subqueries & Nested Queries
•	Window functions (Advanced level)
•	Business-oriented data analysis

📊 **Problem Breakdown & Learning Scope**
🟢 **Easy Level** (Basic SQL + Aggregation)
Focus: Understanding data & simple insights
•	Identify senior-most employee using job hierarchy
•	Find countries with maximum invoices
•	Analyze top invoice values
•	Determine highest revenue-generating city
•	Identify best customer (highest spender)
👉 **Concepts used:**
GROUP BY, ORDER BY, LIMIT, SUM(), COUNT()

🟡 **Moderate Level** (Joins + Filtering + Subqueries)
Focus: Combining multiple tables
•	Extract Rock music listeners with customer details
•	Identify top Rock artists by track count
•	Find songs longer than average duration
👉 **Concepts used:**
INNER JOIN, WHERE, HAVING, AVG(), subqueries

🔴 **Advanced Level** (Complex Analysis + Business Insights)
Focus: Real-world analytical thinking
•	Calculate total amount spent by each customer on each artist
•	Identify most popular genre per country
•	Find top-spending customer per country
👉 **Concepts used:**
CTE (WITH), WINDOW FUNCTIONS (RANK, DENSE_RANK), advanced joins

💡 **Business Insights Derived**
•	Identified high-value customers for targeted marketing
•	Discovered top-performing artists & genres
•	Analyzed geographical revenue trends
•	Helped in planning music festivals in high-revenue cities
•	Enabled country-level personalization strategies

🚀 **Project Outcome**
This project showcases the ability to:
•	Work with real-world relational databases
•	Solve business-driven SQL problems
•	Perform end-to-end data analysis using SQL only
•	Translate raw data into actionable insights
